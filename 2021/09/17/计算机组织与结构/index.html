第2章 计算机的运算方法

2.1 无符号数与有符号数

计算机中的数均存放在寄存器中，通常称寄存器的位数为机器字长。

2.1.1 无符号数

无符号数：寄存器中的每一位都用来存放数值。

2.1.2 有符号数

将符号数字化（0，1化）的数称为机器数，而带“+”，”-“号的数称为真值。

原码、补码、反码、移码都属于机器数。

有符号数：寄存器中的最高位用来标识符号，其余几位用来存放数值。

整数的符号位与数值位之间用逗号（,）隔开，小数的符号位与数值位之间用小数点（.）隔开。

有符号数的表示有四种：

  1. 原码表示法

    一位符号位+绝对值位：如下公式表达含义即为正数最高位为0，负数最高位为1，其余位数用来表示数的绝对值。
    $$
    整数原码{,}{,}{,}{,}[x]_原=\left{
    \begin{aligned}
    0,x && 2^n>x\geq 0 \
    2^n -x && 0\geq x>-2^n \
    \end{aligned}
    \right. {,}{,}{,}{,}{,}{,}{,}{,}{,}x为真值，n为机械数的位数
    $$

    $$
    小数原码{,}{,}{,}{,}[x]_原=\left{
    \begin{aligned}
    x && 1>x\geq 0 \
    1 -x && 0\geq x>-1 \
    \end{aligned}
    \right. {,}{,}{,}{,}{,}{,}{,}{,}{,}x为真值
    $$
    原码表示的优劣：

    简单明了，易于理解真值，却无法直接进行正数与负数的加法运算。

  2. 反码表示法

    正数的反码即为原码，负数的反码为符号位不变，其余按位取反。

  3. 补码表示法

    正数的补码即为原码，负数的补码为其反码末位加1，直观理解为符号位不变，低位的第一个1及之前的0不变，其余位取反。
    $$
    整数补码{,}{,}{,}{,}[x]_补=\left{
    \begin{aligned}
    0,x && 2^n>x\geq 0 \
    2^{n+1} +x && 0>x>-2^n && (mod2^{n+1}) \
    \end{aligned}
    \right. {,}{,}{,}{,}{,}{,}{,}{,}{,}x为真值，n为机械数位数
    $$

    $$
    小数补码{,}{,}{,}{,}[x]_补=\left{
    \begin{aligned}
    x && 1>x\geq 0 \
    2 +x && 0>x\geq -1 && (mod2) \
    \end{aligned}
    \right. {,}{,}{,}{,}{,}{,}{,}{,}{,}x为真值
    $$

    $$
    已知[y]_补求[-y]_补：连同符号位一起按位取反，末位加1。
    $$

补码的思想：理解1—>利用机器数存储位数的有限性，进行周而复始的思想（取余思想）。

补码的思想：理解2—>相反数思想，当计算机运算机器数时，两机械数相加之和为2。

补码表示的优劣:

恰与原码的优劣相反，将所有的加减运算都用机器加法统一了，但却无法直接从补码判断真值。

  1. 移码表示法
    $$
    [x]_移 = 2^n + x {,}{,}{,}{,}(2^n>x\geq-2^n),x表示真值，n为机械数的位数。
    $$
    移码的思想：数轴平移。

    移码中符号位为1表示正数，符号位为0表示复数，这正是

2.2 数的定点表示和浮点表示

2.2.1 定点表示

小数点固定在某一位置，如纯小数和纯整数。

2.2.2 浮点表示

$$
浮点数=尾数\times基数^{阶码}\
N = S \times r^j\
在计算机表示中基数r为2，\frac{1}{2}\leq S<1,此时尾数最高位为1，称为规格化数。
\如此一来尾数为纯小数，阶码也为纯小数。
$$
