1.4 图的连通性

路径：从v0到vk的一个点边交替序列，称为从v0到vk一条路径。$W= v_0e_1v_1e_2v_2\dots e_kv_k$

路径的逆：$W^{-1}$

路径的拼接：$v_ke_k\dots v_le_l$, $WW^1$

路径的长：边的数目称为路经的长。

节：路径W的一个子序列构成的路径称为W的节。$v_ie_i\dots v_je_j$

迹：路径中的边互不相同，称该路径为迹。

路：路径中的点互不相同，称该路径为路。

闭路径：路径中的起点和终点为同一个点，称该路径为闭路径，否则称为开路径。

闭迹：迹中的起点和终点为同一个点，称该迹为闭迹，否则称为开迹。闭迹又称回路。

圈：闭迹中的点除了起点和终点为同一点外，其他再无相同的点，则称该闭迹为圈或者k圈（有k个不同点）。k为偶数称偶圈，k为奇数称奇圈。

简单回路或者简单路径：路径中不包含重复的边。

1.7 平面图

平面图：如果一个图可以被画在平面上，使得它的边仅在顶点处相交，则称这个图为平面图，或者说它是可平面嵌入的。平面图的这样一种画法被称为图的平面表示。平面图的平面嵌入称为平图。

平面图的平面表示将平面划分为多个面(region)，其中包括一个无界面(unbounded region)。

一条连续的，自身不相交的封闭曲线称为Jordon曲线。任何连接Jordon曲线的内点与外点的曲线，必与Jordon曲线相交。

欧拉公式：含有$v$个顶点，$e$条边，$r$个面的平面连通简单图G中存在：$r=e-v+2$。

给定平面连通图$G$，$G$的所有平面嵌入具有相同的面数。

平面图的握手定理：平面图中两倍的边数等于所有面的度数之和。

1.8 图着色

关键词：色数(Chromatic Number)

对一个图$G$的顶点着色，使没有任何两个相邻顶点着上相同的颜色，称为图的正常着色，此过程中所用颜色种数为$k$，称$G$是k可着色的。$k$的最小值即为色数记为，称$G$是$k$色的。

地图：一个没有割边的连通平面图称为地图。

设$G$是一个地图，对$G$的每个平面着色，使得没有任何两个相邻平面着上相同的颜色，称为$G$的正常面着色，此过程中所用的颜色种数为$k$，称为$G$是$k$面可着色的。k的最小值即为$G$的面色数，记为，称$G$是$k$面色的。

3.1 The Basic of Counting

The Sum Rule: Classification.

The key to use this rule correctly is to ensure your classification has no repeated class and no forgot class.

A task can be done in one of $n_1$ ways, in one of $n_2$ ways,…, or in one of nm ways. Where none of the set of $n_i$ ways of doing the task is the same as any of the set of $n_j$ ways, for all pairs $i$ and $j$ with $1≤i<j ≤ m$. Then the number of ways to do the task is $T = n1+n2+…+nm.$

Subtraction Rule:

If a task can be done either in one of $n_1$ ways or in one of $n_2$ ways, then the total number of ways to do the task is $n_1 + n_2$ minus the number of ways $n_c$ to do the task that are common to the two different ways.

Division Rule: There are n/d ways to do a task if it can be done using a procedure that can be carried out in n ways, and for every way w, exactly d of the n ways correspond to way w.

3.2 The Pigeonhole Principle

Definition:

If $k$ is a positive integer and $k+1$ or more objects are placed into $k$ boxes, then there is at least one box containing at least two or more objects of the objects.

The generalized pigeonhole principle:

If $N$ objects are placed into $k$ boxes, then there is at least one box containing at least $\lceil \frac{N}{k}\rceil$ objects.

4.1 Applications of Recurrence Relations

Recurrence Relation:

Using previous information to conclude the current situation. In other words, the current situation is partly defined by the previously happened events but with some variants. Formally:

Definition: A recurrence relation for the sequence {$an$} is an equation that expresses an in terms of one or more of the previous terms of the sequence, namely, $a_0, a_1,\dots, a{n-1}$, for all integers $n$ with $n ≥ n_0$, where $n_0$ is a nonnegative integer.
