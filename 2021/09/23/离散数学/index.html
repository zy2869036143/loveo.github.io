1.3.1 图的表示

  1. 邻接表 Adjacency Lists
  2. 邻接矩阵 Adjacency Matrices
  3. 关联矩阵 Incidence Matrices

无向图的邻接矩阵总是对称的。

带有$n$个顶点的简单图的邻接矩阵具有$n!$种不同表示。

对于无向图来说，邻接矩阵每一行各位置的数字之和代表与该顶点相关联的边数，也代表该顶点的度减去它的环数。

1.3.2 图的同构

存在一个映射将图A的点和相对应的边映射到B中的定点和边。

同构图中的重要不变量：（必要条件）

  • 具有相同的顶点数
  • 具有相同的边数
  • 具有有相同的顶点度
  • 具有相同的连通分支数目及其大小
  • 具有相同长度的简单回路

1.4 图的连通性

路径：从v0到vk的一个点边交替序列，称为从v0到vk一条路径。$W= v_0e_1v_1e_2v_2\dots e_kv_k$

路径的逆：$W^{-1}$

路径的拼接：$v_ke_k\dots v_le_l$, $WW^1$

路径的长：边的数目称为路经的长。

节：路径W的一个子序列构成的路径称为W的节。$v_ie_i\dots v_je_j$

迹：路径中的边互不相同，称该路径为迹。

路：路径中的点互不相同，称该路径为路。

闭路径：路径中的起点和终点为同一个点，称该路径为闭路径，否则称为开路径。

闭迹：迹中的起点和终点为同一个点，称该迹为闭迹，否则称为开迹。闭迹又称回路。

圈：闭迹中的点除了起点和终点为同一点外，其他再无相同的点，则称该闭迹为圈或者k圈（有k个不同点）。k为偶数称偶圈，k为奇数称奇圈。

定理1 若图G中每个顶点的度至少为2，则G中必有圈。

证明1

取图$G$中某一顶点$v_1$，因为$deg(v)\ge 2$，所以$v_1$一定与某一之前未取到的顶点$v_2$相连。同理$v_2$一定与某一之前未取到的顶点$v_3$相连 $\dots\dots $

因为图中顶点个数是有限的，所以伴随上述过程一直进行，必然会取到一个点$v_k$，$v_k$满足以下条件：与$v_k$相连的所有顶点都在{$V | v_i\in V, i=1,2,\dots k-1$}中。此时就产生了圈。

图中两点间存在路径，则称该两点是联通的。若图G中任意两点都是联通的，则称图G是联通的。

连通分支：相当于等价类，将所有联通的点和边划分为一部分。图G的连通分支数常用ω(G)表示。

两点间路径的长称为该两点间的距离。

1.5 欧拉通路与哈密顿通路

关键词：欧拉通路，欧拉回路，欧拉图，半欧拉图；哈密顿回路，哈密顿通路，哈密顿图，半哈密顿图

定理1 连通多重图具有欧拉回路当且仅当它的每个顶点都有偶数度。

定理2 连通多重图具有欧拉通路而无欧拉回路，当且仅当它恰有两个奇数度顶点。其中一个为起始点，另一个为终点。

定理1-1 一个没有孤立顶点的有向多重图含有欧拉回路的充要条件是：该图是弱联通的；每个顶点的入度和出度相等。

定理2-2 一个没有孤立顶点的有向多重图含有欧拉通路而无欧拉回路的充要条件是：该图是弱联通的；除去两个顶点外，每个顶点的入度和出度相等，这两个顶点中一个入度比出度大1，另一个出度比入度大1。

定理3 狄拉克定理 如果G是带有$n$个顶点的连通简单图，其中$n\ge3$，则G有哈密顿回路的充分条件是每个顶点的度都至少为$\frac{n}{2}$。

引理：设G是简单图，顶点u，v是一对不相邻的顶点，则满足$deg(u)+deg(v)>=n$，则G是哈密顿图当且仅当G+uv是哈密顿图。

定理4 奥尔定理 如果G是带n个顶点的连通简单图，其中n>=3，并且对于G中每一对不相邻的顶点u，v来说，都有$deg(u)+deg(v)>=n$，则G有哈密顿回路。

1.7 平面图

平面图：如果一个图可以被画在平面上，使得它的边仅在顶点处相交，则称这个图为平面图，或者说它是可平面嵌入的。平面图的这样一种画法被称为图的平面表示。平面图的平面嵌入称为平图。

平面图的平面表示将平面划分为多个面(region)，其中包括一个无界面(unbounded region)。

一条连续的，自身不相交的封闭曲线称为Jordon曲线。任何连接Jordon曲线的内点与外点的曲线，必与Jordon曲线相交。

欧拉公式：含有$v$个顶点，$e$条边，$r$个面的平面连通简单图G中存在：$r=e-v+2$。

给定平面连通图$G$，$G$的所有平面嵌入具有相同的面数。

平面图的握手定理：平面图中两倍的边数等于所有面的度数之和。

定理1 有$v$个顶点，$e$条边，r个面的平面图G中，n>=3时，$e\le3v-6$。

证明1

因为每个平面的度至少为3，所以：$2e=\sum deg(R) \ge3r$

又因为$r=e-v+2$，所以$e\le3v-6$

定理2 如果G是一个连通的平面简单图，那么G一定有一个度数不大于5的顶点。

如果G的所有顶点度数都大于5，则$ 6v\le\sum deg(v)=2e \le6v-12$

1.8 图着色

关键词：色数(chromatic number)

对一个图$G$的顶点着色，使没有任何两个相邻顶点着上相同的颜色，称为图的正常着色，此过程中所用颜色种数为$k$，称$G$是k可着色的。$k$的最小值即为色数记为，称$G$是$k$色的。

地图：一个没有割边的连通平面图称为地图。

设$G$是一个地图，对$G$的每个平面着色，使得没有任何两个相邻平面着上相同的颜色，称为$G$的正常面着色，此过程中所用的颜色种数为$k$，称为$G$是$k$面可着色的。k的最小值即为$G$的面色数，记为，称$G$是$k$面色的。

平面图的dual graph

四色定理：任何平面图都是4可着色的。

五色定理：任何无自环的平面图都是5可着色的。

2.2 The Pigeonhole Principle

Definition:

If $k$ is a positive integer and $k+1$ or more objects are placed into $k$ boxes, then there is at least one box containing at least two or more objects of the objects.

The generalized pigeonhole principle:

If $N$ objects are placed into $k$ boxes, then there is at least one box containing at least $\lceil \frac{N}{k}\rceil$ objects.

2.5 Generalized Permutations and Combinations

$n$ is the number of objects and $r$ is the number of objects that we want to chose.

In the cases that repetition is not allowed, each object is a unique one, which means each object is distinguished from the others.

But in the cases that repetition is allowed, $n$ can be understood as the different classes of the objects. Each class contains numerous indistinguishable objects whose number is enough for you to extract.

3.1 Applications of Recurrence Relations

神奇的斐波那契数列：

  1. 最后一位数字，每60个数循环一次；最后两位数字，每300个数循环一次；最后三位数字，每1500个数循环一次；最后四位数字，每15000个数循环一次；最后五位数字，每150000个数循环一次。

  2. 每第三个数可被2整除，每第四个数可被3整除，每第五个数可被5整除，每第六个数可被8整除，等等。这些除数本身也构成斐波那契数列。

  3. 在斐波那契数列中，有素数：2，3，5，13，89，233，1597，28657，514229，433494437，2971215073，99194853094755497，……除了n=4之外，所有的斐波那契素数的序号都是素数。

Recurrence Relation: Using previous information to conclude the current situation. In other words, the current situation is partly defined by the previously happened events but with some variants.
